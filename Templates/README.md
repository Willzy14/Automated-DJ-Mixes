# Templates

Place your Ableton Live 12 base template here.

To create one:
1. Open Ableton Live 12
2. Create an empty session at 128 BPM
3. Add 10-12 empty audio tracks
4. Save As to this folder (e.g. `base_template.als`)

The pipeline will decompress this template, study the XML structure, and use it as the base for all generated sessions. A real template is required before the ALS generator module can be built.
